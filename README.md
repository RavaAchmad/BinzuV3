## 🖥️ **Installation**

### **For Windows/VPS/RDP:**

**1. Install Required Software:**
- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/) (v20+)
- [ImageMagick](https://imagemagick.org/)
- [FFmpeg](https://ffmpeg.org/) (Add to PATH)

**2. Deploy Bot:**
```bash
git clone https://github.com/XM4ZE/XMYULA-MD
cd XMYULA-MD
npm install
npm start
```
